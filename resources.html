What is SAR?

Synthetic Aperture Radar (SAR) is a type of radar technology used to create highly detailed images of objects, landscapes, and surfaces, particularly for remote sensing applications. Unlike traditional radar, SAR can produce high-resolution images regardless of weather conditions, time of day, or atmospheric visibility, making it extremely valuable for a variety of applications.
SAR image is an active microwave-based imaging. Being the wavlength longer, images are not affected by clouds, haze, and other meteorological conditions that otherwise affect visible images.
SAR images mainly reveal the structural properties of the target scene, such as dielectric properties, surface roughness, and moisture content. Therefore, from SAR images it is possible to detect various objects based on their surface features containing rich spatial information

Breakdown of SAR

Spatial Resolution and Antenna Size:

  • Spatial Resolution: This refers to the ability of a radar system to distinguish between two closely spaced objects on the ground. Higher spatial resolution means finer detail in the radar images.
  • Wavelength and Antenna Length: The spatial resolution of radar is directly linked to the ratio of the radar wavelength to the length of the antenna. For a given wavelength, a longer antenna is required to achieve higher spatial resolution.

The electromagnetic spectrum with microwave bands inset

Synthetic Aperture Solution:

  • Impractical Antenna Size: Building and deploying a 4,250-meter-long antenna in space is not feasible.
  • Synthetic Aperture: SAR overcomes this limitation by using a much shorter antenna and moving it across the target area. The radar captures multiple signals over time as it moves, and these signals are then combined computationally to simulate a much larger antenna. This "synthetic" aperture allows for high-resolution images without the need for a physically large antenna.

RBG-LAB Coversion Model

RGB
  • Stands for: Red, Green, Blue
  • Nature: Additive color model, where colors are created by combining different intensities of the primary colors (red, green, and blue).
  • Range: Typically, each channel (R, G, B) can have values from 0 to 255 in 8-bit systems.
  • Usage: Commonly used in digital displays like monitors, TVs, and cameras. It is suitable for devices that emit light.
  • Representation: RGB is represented as a 3-tuple (R, G, B), where each value specifies the intensity of red, green, and blue.
    • Example: (255, 0, 0) represents pure red.

RBG COLOR PALLETTE

LAB
  • Stands for: Lightness (L), A (green to red), and B (blue to yellow)
  • Nature: Perceptually uniform color model, meaning changes in LAB values are intended to correspond more closely to human perception of color differences.
  • Usage: Often used in color correction, printing, and color science since it's designed to be device-independent and closer to how humans perceive color.
  • Representation:
    • L (Lightness): Represents brightness, ranging from 0 (black) to 100 (white).
    • A: Represents the green-red component, where negative values indicate green and positive values indicate red.
    • B: Represents the blue-yellow component, where negative values indicate blue and positive values indicate yellow.

HOW THIS MODEL WORKS?

1. Using RGB LAB Conversion
A ImgeNet Database can be used a source for RGB color space data and this RBG images need to be converted into LAB color space

2. Extracting the L Channel (Grayscale Image)
Once an RGB image is converted into LAB format, the L channel is isolated. This L channel acts like a grayscale image because it only contains intensity information (brightness) without any color.

3. Training the DL network using L channel
This L channel is used as the input for the neural network to train the model. Since it's effectively a grayscale image, the model can learn pixel intensity and brightness information.

4. Combine the L chnnnel and Predicted ab chnnel data
Once the model has been trained, for any new grayscale (L channel) input image, the network can predict the missing A and B channels. These channels contain the color information that will be combined with the L channel to form the color image.

5. Convert the LAB image into RGB image
The LAB image is then converted back to RGB to produce a fully colorized image.
